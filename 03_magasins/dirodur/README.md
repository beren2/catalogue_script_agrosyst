
## Le projet DiRoDur

### Résumé

DiRoDur valorise les données des fermes DEPHY (grandes cultures avec/sans élevage) par
l’étude des liens entre la diversification des successions culturales et (i) le niveau de recours
aux pesticides, et (ii) les performances économiques, sociales et environnementales. DiRoDur
proposera un indicateur de diversification prédictif des indicateurs IFT/HRI, et évaluera un
scénario de diversification généralisée en termes d’impacts sur les volumes de production et
la balance commerciale de la France.

## Le magasin DiRoDur
### Ambitions du magasin de données

Les intérêts de mettre à disposition un magasin de données unique pour les utilisateurs du projet et que celui-ci soit co-construit par un comité d'experts des données Agrosyst sont :  

* de garantir plus d’homogénéité dans les données (plus de clarté dans les échanges entre les participants)

* de diminuer le temps investi dans le traitement par chacun des participants (travail effectué une seule fois par le groupe d’experts)

* d’augmenter la fiabilité des choix effectués sur les données (groupe d’experts maîtrisant les données)

* de pérenniser sur le long terme la base commune (magasin DiRoDur disponible sur Datagrosyst)
